## 补充
- functioan 后 加 * 生成器函数: 调用生成器函数会产生一个生成器（generator）。生成器拥有的最重要的方法是 next()，用来迭代
- yield: 关键字用来暂停和继续执行一个生成器函数。当外部调用生成器的 next() 方法时，yield 关键字右侧的表达式才会执行。执行结果会转化为一个对象（包含两个属性， value 和 done），作为 next() 方法的返回值。
